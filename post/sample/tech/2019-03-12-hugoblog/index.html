Hugo

安装、运行

  • hugo.exe
  • https://github.com/spf13/hugo/releases
  • 环境变量Temp

生成站点

hugo new site D:\B002.资料\A01.个人站点\terrylu

生成POST

hugo new about.md
hugo new post/first.md

安装主题

cd themes
git clone https://github.com/spf13/hyde.git
git clone https://github.com/Fastbyte01/KeepIt.git
  • https://github.com/gohugoio/hugoThemes
  • https://themes.gohugo.io/

运行

hugo server

hugo server

hugo server --theme=hugo-pacman-theme --buildDrafts

http://localhost:1313+

部署

hugo  --baseUrl="https://terryluwenbinbin.github.io/"

常用命令
